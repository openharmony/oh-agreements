# OpenHarmony 迁移公告

**亲爱的鸿蒙开发者们：**

OpenHarmony 是开放原子开源基金会旗下众多的开源项目之一，致力于构建开放、共享的技术生态，并为用户提供更好的产品和服务。为了进一步提升用户体验和数据安全性，OpenHarmony项目群工作委员会（下文简称“我们”或“工委会”）计划将 OpenHarmony 项目数据迁移至 GitCode 平台（gitcode.com）。

## 一、数据迁移计划

**自本公告中分批迁移的项目正式宣布迁移之日起，我们将正式启动 OpenHarmony 项目的数据迁移工作。** 已经签署授权协议的用户的数据将迁移至 GitCode平台（gitcode.com），未签署授权协议或撤销授权协议的用户的数据不会进行迁移。数据迁移完成后，Gitee 平台上的所有数据仍然保留，用户可以继续使用 Gitee 账号登录 Gitee 平台进行查看。

- **1. 未签署授权协议或已撤销授权的用户，其数据将不予迁移。**
- **2. 授权期限截止日期为本公告中分批迁移的项目正式宣布迁移之日的 90 天后。逾期未授权的用户，将无法再申请数据迁移。**
- **3. 数据迁移完成后，Gitee 平台上的原有项目数据将继续保留。**

我们将在整个迁移过程中确保数据的完整性与安全性，同时为您提供详尽的指导和支持，帮助您快速适应新平台的操作环境。

## 二、如何联系我们

如果您有任何疑问或投诉，您可以发送邮件至 kefu@gitcode.com 邮箱或电话：400-6868-951 联系我们。一般情况下，我们将在15个工作日内回复。

如果您同意此次数据迁移，您可于本公告中分批迁移的项目正式宣布迁移之日的 90天内通过在 GitCode 签署协议来确认您的选择。如果您不同意迁移，则不要签署迁移协议。

_[点击签署并提交协议>>>](https://gitcode.com/openharmony/oh-agreements/issues/create/choose)_

感谢您对鸿蒙 OpenHarmony 项目的支持与理解！
